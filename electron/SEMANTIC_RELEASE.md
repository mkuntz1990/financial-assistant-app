# Semantic Release configuration for Financial Assistant App
# Run this from the electron directory:
#   npx semantic-release
#
# This will:
# - Analyze commit messages for version bump (major, minor, patch)
# - Update package.json and CHANGELOG.md
# - Create a GitHub release with changelog
# - Push changes to GitHub
#
# To use, make sure your commits follow Conventional Commits:
#   feat: for new features
#   fix: for bug fixes
#   BREAKING CHANGE: for breaking changes
